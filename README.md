# Laravel Tanstack

Merupakan aplikasi CRUD sederhanan dengan react sebagai frontend dan laravel sebagai backend untuk menghandel api dengan stackmanagement redux dan penggunaan tanstack query untuk fetching API dari backend.

## Laravel

backend api dibangun menggunakan laravel versi 12.0 dengan feature :

- Register
- Login
- JWT Auth dengan Sanctum (token dengan Bearer)
- fetch Data user (register,login,show)
- fetch Data Product (index,create,show,update,delete)

## React

react sebagai frontend untuk menampilkan visual pada aplikasi dengan feature :

- axios
- tanstack query
- redux (statemangement)
